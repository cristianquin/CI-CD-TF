#CI/CD-TF 🚀

Proyecto enfocado en la implementación de un flujo CI/CD utilizando Terraform como herramienta de Infrastructure as Code (IaC).

El objetivo principal es automatizar la gestión y despliegue de infraestructura mediante pipelines, permitiendo ejecutar procesos repetibles, controlados y versionados.

#Objetivos del proyecto
Automatizar procesos de integración y despliegue.
Gestionar infraestructura mediante código.
Aplicar prácticas DevOps para mejorar la entrega de cambios.
Reducir configuraciones manuales y errores humanos.
#Tecnologías utilizadas
Terraform
CI/CD Pipeline
GitHub Actions
Cloud Infrastructure
Git
#Flujo del proyecto
Un cambio es enviado al repositorio.
El pipeline ejecuta validaciones automáticas.
Terraform analiza los cambios necesarios.
Se genera el plan de infraestructura.
Se ejecuta el despliegue según la configuración definida.
#Beneficios

✅ Infraestructura versionada
✅ Automatización de procesos
✅ Mayor trazabilidad de cambios
✅ Despliegues reproducibles
✅ Base escalable para ambientes productivos
